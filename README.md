# TripAdvisor Clone

## 📌 Overview
This project is a **TripAdvisor Clone** built using **HTML and CSS**. It recreates the basic design and layout of the TripAdvisor homepage with a clean and responsive user interface.

## 🚀 Features
- Responsive Navigation Bar
- Search Section
- Categories and Menu Bar
- Recommended Destinations Section
- Travel Cards with Images and Details
- Fully Responsive Design

## 🛠️ Technologies Used
- HTML5
- CSS3
- Flexbox
- Media Queries

## 📂 Project Structure
```
TripAdvisor-Clone/
│
├── index.html
├── style.css
├── images/
└── README.md
```

## 🔧 How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/tripadvisor-clone.git
   ```
2. Open the project folder.
3. Run `index.html` in your browser.

This project was created to practice front-end development skills by replicating a real-world website design.
